The problem & the idea

Agents are black boxes

  • They make judgment calls with no shared memory of how past ones went.
  • They can't explain why — or trace a decision across the agents that made it.
  • So they drift: inconsistent, un-auditable, hard to trust.

A hosted decision memory

  • Recall precedent · record a decision · ask why
  • One SKILL.md, 3 endpoints, no auth — nothing to install
  • Semantic recall (embeddings) + a causal graph (Neo4j)

What you just saw

Consistent decisions + an auditable why

  • Semantic precedent recall keeps agents consistent with past practice.
  • Every decision is a trace — inputs, reasoning, tool calls, outcome.
  • Causal chains link decisions across agents: follow why back to the root.
  • A fresh agent used it end-to-end from the SKILL.md alone — verified in the graph DB.
